# 项目：极简胶囊衣橱 (Minimalist Wardrobe) App

## 1. 核心目标
开发一款面向大众用户的 SaaS 模式手机 App。
核心功能：用户注册登录、拍照上传衣物、云端存储、自动抠图（预留接口）、穿搭组合。

## 2. 技术栈确认 (必须严格遵守 .cursorrules)
- **前端框架:** React Native (Expo SDK 50+)
- **路由:** Expo Router (文件路由)
- **UI 库:** NativeWind (Tailwind CSS) + Lucide React Native (图标)
- **后端/数据库:** Supabase (Postgres + Auth + Storage)
- **状态管理:** React Query (TanStack Query)
- **核心语言:** TypeScript

## 3. 核心数据结构 (Interfaces)
请设计以下 Interface，并确保包含 `user_id` 以实现数据隔离：
- `User` (用户概览)
- `ClothingItem` (衣物：包含图片URL、分类、季节、穿搭次数等)
- `Outfit` (穿搭：包含衣物ID数组、穿搭日期、场景等)
